# WeatherWeb
The website show current weather and forecast weather by enter the city name.
+ Using jsp, servlet.
+ Using OpenWeatherAPI 

#Environment:
+ jdk 8
+ Kotlin/Java Library for OpenWeatherMap.org Weather APIs: owm-japis-2.5.3.0.jar
